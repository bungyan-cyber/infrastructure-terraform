# Infrastructure as Code with Terraform

This project sets up a basic infrastructure on AWS using Terraform.

## Requirements

- Terraform
- AWS Account

## Infrastructure Components

- A VPC
- One public subnet
- An EC2 instance running Ubuntu
- An S3 bucket
